## 2. Missing / Enterprise Features (ðŸ”´ The Gap)
To achieve a "Full Fledged" Enterprise HRMS, the following modules are required:

### Priority 1: Payroll & Compensation
*   **Salary Structure**: Definition of Basic, HRA, DA, Allowances for each grade.
*   **Payslip Generation**: Automatic PDF generation for employees.
*   **Tax/Deductions**: managing PF, Tax deductions.

### Priority 2: Document Management
*   **Digital Locker**: Secure storage for Contracts, ID Proofs, Offer Letters.
*   **Expiry Alerts**: Notifications for expiring documents (visas, certifications).

### Priority 3: Performance Management
*   **Appraisals**: Quarterly/Yearly review cycles.
*   **Goals/OKRs**: Setting and tracking employee objectives.
*   **Self-Assessment**: Workflows for employees to rate themselves.

### Priority 4: Onboarding & Offboarding
*   **Checklists**: Automated tasks for IT, Admin, and HR when a user joins/leaves.
*   **Asset Management**: Tracking laptops, phones, and licenses assigned to users.

### Priority 5: Shift & Roster Management
*   **Shift Definitions**: Morning, Evening, Night shifts.
*   **Rostering**: Assigning shifts to employees on a weekly basis.

---

## 3. Recommended Roadmap

1.  **Phase 1 (Immediate)**: Implement **Payroll System**. It is the highest value add for any organization.
2.  **Phase 2**: Add **Document Management** to go paperless.
3.  **Phase 3**: Build **Performance Goals** module.
